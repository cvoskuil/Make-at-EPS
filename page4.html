MakerBot Replicator 2

Operating Instructions

1. Find an object to print. Thingiverse is a good place to start. You can also make your own object using a CAD program.

2. Use MakerBot Desktop to view your object file and prepare it for printing. When you are ready, export the print to a SD card as a *.X3G file.

3. Ensure that the filament color you wish to use is connected to the Replicator 2.

3. Connect the SD card to the Replicator 2.

4. Select "Build from SD" and select your print from the list.

5. Start the print.
